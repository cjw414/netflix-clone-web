# netflix-clone-web
React 기반 웹 개발 역량 강화를 위해 넷플릭스 서비스를 클론 구현한 학습 프로젝트

1. AWS S3/EC2 배포 및 테스트
 - 클라우드 배포 및 기본 테스트를 수행합니다.

2. Dockerfile 작성 및 이미지 빌드
 - 배포 환경 위한 도커 이미지를 구성합니다.

3. TypeScript 타입 정의
 - props, API 타입 안정화 수행합니다.

4. 인터랙션 추가 (모달, 슬라이더 등)
 - Hover, 클릭 인터페이스 등의 인터랙션을 추가합니다.

5. UI/UX 구성 및 페이지 디자인
 - 포스터, 리스트, 디테일 등 핵심 UI를 구현합니다.
